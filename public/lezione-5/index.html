Lezione 5 - JavaScript, TypeScript e React

Tecnologie fondamentali per lo sviluppo web moderno

Argomento: JavaScript ES6+, TypeScript e React - le tecnologie essenziali per costruire applicazioni web moderne e scalabili

Descrizione

Questa lezione introduce i fondamenti di JavaScript, TypeScript e React. Inizieremo con le basi di JavaScript: variabili, funzioni, oggetti, classi e iterazioni. Poi vedremo come TypeScript aggiunge il controllo dei tipi a questi concetti, rendendo il codice più sicuro. Infine, React ci permetterà di costruire interfacce utente interattive con componenti riutilizzabili.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Dichiarare variabili con const e let
  • Creare e usare funzioni, oggetti e classi in JavaScript
  • Iterare su array con for, while e forEach
  • Aggiungere tipi a variabili, funzioni, array, oggetti e classi con TypeScript
  • Comprendere i vantaggi del controllo dei tipi
  • Costruire componenti React riutilizzabili
  • Gestire lo stato con useState
  • Gestire eventi e interazioni utente
  • Integrare TypeScript con React
