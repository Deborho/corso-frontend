Lezione 5 - JavaScript, TypeScript e React

Tecnologie fondamentali per lo sviluppo web moderno

Argomento: JavaScript ES6+, TypeScript e React - le tecnologie essenziali per costruire applicazioni web moderne e scalabili

Descrizione

Questa lezione introduce le tre tecnologie fondamentali per lo sviluppo web moderno. JavaScript ES6+ rappresenta l'evoluzione del linguaggio di programmazione più utilizzato nel web, con funzionalità moderne che migliorano produttività e leggibilità del codice. TypeScript estende JavaScript aggiungendo un sistema di tipi che aiuta a prevenire errori e migliorare la manutenibilità del codice. React è la libreria più popolare per costruire interfacce utente interattive basate su componenti riutilizzabili.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Utilizzare le funzionalità moderne di JavaScript ES6+ (arrow functions, destructuring, spread/rest, promises, async/await)
  • Comprendere i tipi fondamentali di TypeScript e come applicarli
  • Creare interfacce con TypeScript per definire la struttura dei dati
  • Costruire componenti React riutilizzabili
  • Gestire lo stato delle applicazioni con useState
  • Implementare effetti collaterali con useEffect
  • Gestire eventi e interazioni utente
  • Integrare TypeScript con React per applicazioni type-safe
  • Costruire un progetto completo utilizzando tutte e tre le tecnologie
