Lezione 1 - Introduzione

Fondamenti di HTML e CSS

Durata: 2 ore

Argomento: Introduzione e approfondimento di HTML e CSS, le basi del linguaggio di markup e styling per il web

Descrizione

Questa lezione introduce i due linguaggi fondamentali per lo sviluppo web: HTML (HyperText Markup Language) e CSS (Cascading Style Sheets). HTML definisce la struttura e il contenuto delle pagine web, mentre CSS controlla la presentazione visuale e il layout degli elementi.

In questa lezione apprenderemo come creare documenti HTML validi seguendo gli standard HTML5 e come applicare stili CSS per controllare l'aspetto degli elementi, creare layout e migliorare l'esperienza utente.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Comprendere la struttura base di un documento HTML5
  • Utilizzare correttamente i tag semantici HTML5
  • Creare e formattare contenuti testuali (titoli, paragrafi, liste)
  • Inserire link ipertestuali e immagini
  • Applicare stili CSS agli elementi HTML
  • Utilizzare i selettori CSS per stilizzare elementi specifici
  • Comprendere e applicare il Box Model
  • Gestire il posizionamento degli elementi
  • Creare layout flessibili utilizzando Flexbox
  • Creare layout bidimensionali utilizzando CSS Grid
  • Sviluppare una semplice pagina web completa e stilizzata
