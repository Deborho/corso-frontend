Argomento: Introduzione e approfondimento di HTML, le basi del linguaggio di markup per il web

Descrizione

Questa lezione introduce HTML (HyperText Markup Language), il linguaggio fondamentale per lo sviluppo web. HTML definisce la struttura e il contenuto delle pagine web attraverso l'uso di tag ed elementi.

In questa lezione apprenderemo come creare documenti HTML validi seguendo gli standard HTML5, utilizzare tag semantici, creare form interattivi e strutturare correttamente il contenuto delle pagine web.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Comprendere la struttura base di un documento HTML5
  • Utilizzare correttamente i tag semantici HTML5
  • Creare e formattare contenuti testuali (titoli, paragrafi, liste)
  • Inserire link ipertestuali e immagini
  • Creare form HTML per la raccolta dati
  • Utilizzare i diversi tipi di input HTML5
  • Implementare la validazione HTML5 nei form
