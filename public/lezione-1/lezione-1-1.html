Lezione 1 - Struttura Base HTML

Fondamenti del linguaggio di markup per il web

1. Introduzione ad HTML

HTML (HyperText Markup Language) è il linguaggio di markup standard utilizzato per creare pagine web. Non è un linguaggio di programmazione, ma un linguaggio di marcatura che definisce la struttura e il contenuto di un documento web attraverso l'uso di tag ed elementi.

1.1 Sintassi dei Tag

Gli elementi HTML sono definiti da tag racchiusi tra parentesi angolari. La maggior parte degli elementi richiede un tag di apertura e uno di chiusura:

<tagname>contenuto</tagname>

Alcuni elementi sono auto-chiudenti:

<br>
<hr>

2. Struttura Base del Documento

Ogni documento HTML5 valido deve seguire questa struttura fondamentale:

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titolo della Pagina</title>
</head>
<body>
    <!-- Contenuto visibile -->
</body>
</html>

2.1 Elementi della Struttura

Elemento Descrizione
<!DOCTYPE html> Dichiara il tipo di documento HTML5
<html> Elemento radice del documento
<head> Contiene metadati e informazioni sul documento
<meta charset> Definisce la codifica dei caratteri
<meta viewport> Controlla il rendering su dispositivi mobili
<title> Titolo della pagina visualizzato nel browser
<body> Contiene tutto il contenuto visibile

3. Tag Semantici HTML5

HTML5 introduce elementi semantici che descrivono il loro significato in modo chiaro. L'utilizzo di questi elementi migliora l'accessibilità, la manutenibilità del codice e l'indicizzazione da parte dei motori di ricerca.

3.1 Principali Tag Semantici

Tag Utilizzo
<header> Intestazione di una pagina o sezione
<nav> Menu di navigazione
<main> Contenuto principale del documento
<section> Sezione tematica di contenuto
<article> Contenuto autonomo e indipendente
<aside> Contenuto correlato ma secondario
<footer> Piè di pagina di una sezione o documento

3.2 Esempio di Struttura Semantica

<body>
    <header>
        <h1>Titolo del Sito</h1>
        <nav>
            <!-- Menu di navigazione -->
        </nav>
    </header>
    
    <main>
        <section>
            <h2>Sezione 1</h2>
            <p>Contenuto...</p>
        </section>
        
        <section>
            <h2>Sezione 2</h2>
            <p>Contenuto...</p>
        </section>
    </main>
    
    <footer>
        <p>Informazioni di contatto</p>
    </footer>
</body>

Titolo del Sito

Sezione 1

Contenuto...

Sezione 2

Contenuto...

Informazioni di contatto

4. Tag di Testo

4.1 Titoli (Headings)

HTML fornisce sei livelli di titoli, da <h1> a <h6>, dove h1 rappresenta il titolo più importante:

<h2>Sottotitolo</h2>
<h3>Sottosezione</h3>
<h4>Titolo di Livello 4</h4>
<h5>Titolo di Livello 5</h5>
<h6>Titolo di Livello 6</h6>

5. Liste

5.1 Liste Non Ordinate

<ul>
    <li>Primo elemento</li>
    <li>Secondo elemento</li>
    <li>Terzo elemento</li>
</ul>
  • Primo elemento
  • Secondo elemento
  • Terzo elemento

5.2 Liste Ordinate

<ol>
    <li>Primo passo</li>
    <li>Secondo passo</li>
    <li>Terzo passo</li>
</ol>
  1. Primo passo
  2. Secondo passo
  3. Terzo passo

7. Multimedia

7.1 Tag Video

Il tag <video> permette di incorporare video nella pagina web:

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Il tuo browser non supporta il tag video.
</video>

7.2 Tag Iframe

Il tag <iframe> permette di incorporare contenuti esterni, come mappe di Google Maps:

7.3 Tag Img

Il tag <img> permette di inserire immagini nella pagina web:

<img src="esempio.jpg" alt="Descrizione dell'immagine" width="300" height="200">

8. Esempio Completo: Pagina Web Base

Di seguito un esempio completo di pagina HTML che utilizza tutti gli elementi presentati:

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Corso Web Frontend</title>
</head>
<body>
    <header>
        <h1>Corso di Web Frontend</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#corso">Il Corso</a></li>
                <li><a href="#contatti">Contatti</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section id="home">
            <h2>Benvenuti</h2>
            <p>Questo corso introduce i <strong>fondamenti dello sviluppo web</strong> utilizzando HTML, CSS e JavaScript.</p>
            <img src="banner.jpg" alt="Banner del corso">
        </section>
        
        <section id="corso">
            <h2>Programma del Corso</h2>
            <p>Il corso è strutturato in diverse lezioni che coprono:</p>
            <ol>
                <li>HTML e struttura delle pagine web</li>
                <li>CSS per lo styling e il layout</li>
                <li>JavaScript per l'interattività</li>
                <li>Framework moderni come React</li>
            </ol>
            
            <h3>Requisiti</h3>
            <ul>
                <li>Conoscenze informatiche di base</li>
                <li>Computer con accesso a internet</li>
                <li>Editor di testo installato</li>
            </ul>
        </section>
        
        <section id="contatti">
            <h2>Contatti</h2>
            <p>Per informazioni:</p>
            <p><a href="mailto:info@corsoweb.com">info@corsoweb.com</a></p>
            <p>Tel: +39 123 456 7890</p>
        </section>
    </main>
    
    <footer>
        <p>&copy; 2025 Corso Web Frontend.</p>
    </footer>
</body>
</html>
