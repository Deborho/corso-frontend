Breakpoint Responsive

Mobile: 0-767px

Layout a colonna singola, menu verticale, sidebar sotto il contenuto principale.

Tablet: 768-1023px

Grid a 2 colonne, menu orizzontale con wrap, sidebar ridotta.

Desktop: 1024-1399px

Grid a 4 colonne, layout completo con sidebar laterale.

Large Desktop: 1400px+

Container più largo, grid a 4 colonne con spazi maggiori.
