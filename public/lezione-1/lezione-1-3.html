Lezione 1.3 - CSS Base

Cascading Style Sheets: Fondamenti dello styling web

1. Introduzione a CSS

CSS (Cascading Style Sheets) è il linguaggio utilizzato per definire la presentazione visuale di documenti HTML. Mentre HTML definisce la struttura e il contenuto, CSS controlla l'aspetto, il layout e la formattazione degli elementi.

CSS opera secondo il principio della cascata: le regole più specifiche hanno precedenza su quelle più generiche, e le regole definite successivamente possono sovrascrivere quelle precedenti.

2. Metodi di Inserimento CSS

2.1 CSS Inline

Lo stile viene applicato direttamente nell'elemento HTML tramite l'attributo style:

<p style="color: blue; font-size: 16px;">Testo con stile inline</p>

Testo con stile inline

Questo metodo ha la massima specificità ma è sconsigliato per la manutenzione del codice.

2.2 CSS Interno

Lo stile viene definito all'interno del tag <style> nella sezione <head>:

<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>

2.3 CSS Esterno (Usato principalmente)

Lo stile viene definito in un file separato con estensione .css e collegato al documento HTML:

<head>
    <link rel="stylesheet" href="stile.css">
</head>

Questo è il metodo preferito perché permette di riutilizzare gli stili su più pagine e mantiene una separazione tra contenuto e presentazione.

3. Sintassi CSS

Una regola CSS è composta da un selettore e da un blocco di dichiarazioni:

selettore {
    proprietà: valore;
    proprietà: valore;
}

/* Esempio */
p {
    color: #333;
    font-size: 16px;
    line-height: 1.6;
}

4. Selettori CSS

4.1 Selettori di Base

Selettore Sintassi Descrizione
Elemento p { } Seleziona tutti gli elementi di quel tipo
Classe .nome { } Seleziona elementi con class="nome"
ID #nome { } Seleziona l'elemento con id="nome"
Universale * { } Seleziona tutti gli elementi
Multiplo h1, h2, h3 { } Seleziona più elementi

4.2 Esempi di Selettori

/* Selettore di elemento */
h1 {
    color: #2c3e50;
}

/* Selettore di classe */
.evidenziato {
    background-color: yellow;
}

/* Selettore di ID */
#intestazione {
    font-size: 24px;
}

/* Selettori multipli */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

/* Selettore discendente */
article p {
    text-indent: 20px;
}

/* Selettore figlio diretto */
nav > ul {
    list-style: none;
}

5.3 Box Model

Il Box Model è il concetto fondamentale per comprendere il layout in CSS. Ogni elemento è rappresentato come una scatola rettangolare composta da:

  • Contenuto (content)
  • Padding (spaziatura interna)
  • Border (bordo)
  • Margin (margine esterno)
/* Margini */
margin: 20px;                    /* tutti i lati */
margin: 10px 20px;               /* verticale | orizzontale */
margin: 10px 20px 15px 25px;    /* top | right | bottom | left */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 15px;
margin-left: 25px;

/* Padding */
padding: 20px;
padding: 10px 20px;
padding: 10px 20px 15px 25px;

/* Bordi */
border: 1px solid #ccc;
border-width: 2px;
border-style: solid;
border-color: #333;
border-radius: 5px;               /* angoli arrotondati */

/* Larghezza e altezza */
width: 300px;
height: 200px;
max-width: 100%;
min-height: 400px;

5.4 Display e Visibilità

/* Tipo di visualizzazione */
display: block;           /* elemento a blocco */

display: inline-block;    /* ibrido */
display: none;            /* nasconde l'elemento */

/* Visibilità */
visibility: hidden;       /* nasconde ma mantiene lo spazio */
visibility: visible;

/* Opacità */
opacity: 0.5;            /* da 0 (trasparente) a 1 (opaco) */

7. Flexbox Base

Flexbox è un sistema di layout moderno che permette di creare layout flessibili e responsive con facilità.

/* Contenitore flex */
.container {
    display: flex;
    flex-direction: row;        /* row | column */
    justify-content: center;    /* allineamento orizzontale */
    align-items: center;        /* allineamento verticale */
    gap: 20px;                  /* spazio tra elementi */
}

/* Elementi flex */
.item {
    flex: 1;                    /* cresce proporzionalmente */
    flex-shrink: 0;             /* non si riduce */
    flex-basis: 200px;          /* larghezza di base */
}

8. CSS Grid Layout

CSS Grid è un sistema di layout bidimensionale potente che permette di creare layout complessi con righe e colonne. A differenza di Flexbox che è principalmente monodimensionale, Grid gestisce sia righe che colonne contemporaneamente.

8.1 Proprietà del Container Grid

/* Container Grid */
.container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;     /* definisce 3 colonne */
    grid-template-rows: auto 1fr auto;       /* definisce 3 righe */
    gap: 20px;                               /* spazio tra celle */
    grid-template-areas:                     /* definisce aree nominate */
        "header header header"
        "sidebar main aside"
        "footer footer footer";
}

/* Layout responsive con auto-fit */
.responsive-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 15px;
}

8.2 Proprietà degli Elementi Grid

/* Posizionamento elementi */
.item {
    grid-column: 1 / 3;          /* occupa dalla colonna 1 alla 3 */
    grid-row: 1 / 2;             /* occupa dalla riga 1 alla 2 */
}

/* Usando span */
.wide-item {
    grid-column: span 2;         /* si estende su 2 colonne */
}

.tall-item {
    grid-row: span 3;            /* si estende su 3 righe */
}

/* Usando aree nominate */
.header {
    grid-area: header;
}

8.3 Principali Differenze tra Grid e Flexbox

  • Flexbox: Layout monodimensionale (righe O colonne), ideale per componenti e piccole sezioni
  • Grid: Layout bidimensionale (righe E colonne), ideale per layout completi di pagina
  • Flexbox: Gli elementi si adattano al contenuto
  • Grid: Si definisce una griglia rigida che gli elementi riempiono

10. Variabili CSS (Custom Properties)

Le variabili CSS, chiamate anche custom properties, permettono di memorizzare valori riutilizzabili definiti una sola volta e utilizzati in tutto il foglio di stile. Questo migliora la manutenibilità del codice e facilita la creazione di temi.

10.1 Definizione delle Variabili

/* Definizione globale in :root */
:root {
    --primary-color: #3498db;
    --secondary-color: #2ecc71;
    --spacing-medium: 20px;
    --border-radius: 8px;
    --font-main: Arial, sans-serif;
}

/* Definizione locale in un elemento specifico */
.card {
    --card-padding: 30px;
    padding: var(--card-padding);
}

10.2 Utilizzo delle Variabili

/* Utilizzo con var() */
.button {
    background-color: var(--primary-color);
    padding: var(--spacing-medium);
    border-radius: var(--border-radius);
    font-family: var(--font-main);
}

/* Valore di fallback */
.element {
    color: var(--text-color, black);  /* usa black se --text-color non è definita */
}

10.3 Vantaggi delle Variabili CSS

  • Manutenibilità: Cambia un valore in un unico punto
  • Coerenza: Garantisce uniformità in tutto il progetto
  • Temi dinamici: Permette di cambiare temi facilmente ridefinendo le variabili
  • DRY (Don't Repeat Yourself): Evita la ripetizione di valori hardcoded
  • Leggibilità: Nomi significativi invece di valori numerici o esadecimali

11. Media Query - Layout Responsive e Stampa

Le media query permettono di applicare stili CSS in base alle caratteristiche del dispositivo o del media di output. Sono fondamentali per creare layout responsive che si adattano a schermi di diverse dimensioni e per ottimizzare il contenuto per la stampa.

11.1 Sintassi Base

/* Media query per schermi */
@media screen and (max-width: 768px) {
    /* Stili applicati quando la larghezza è <= 768px */
    .container {
        width: 100%;
        padding: 10px;
    }
}

/* Media query per stampa */
@media print {
    /* Stili applicati solo quando si stampa */
    nav, footer {
        display: none;
    }
}

11.2 Breakpoint Comuni per Layout Responsive

/* Mobile First Approach */
/* Mobile: stili base (0-767px) */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet: 768px e superiori */
@media screen and (min-width: 768px) {
    .container {
        width: 750px;
        padding: 20px;
    }
}

/* Desktop: 1024px e superiori */
@media screen and (min-width: 1024px) {
    .container {
        width: 1000px;
    }
}

/* Large Desktop: 1400px e superiori */
@media screen and (min-width: 1400px) {
    .container {
        width: 1200px;
    }
}

11.3 Media Query per la Stampa

@media print {
    /* Nascondi elementi non necessari */
    nav, .sidebar, .no-print, button {
        display: none !important;
    }
    
    /* Ottimizza per la stampa */
    body {
        font-size: 12pt;
        color: black;
        background: white;
    }
    
    /* Mostra URL dei link */
    a::after {
        content: " (" attr(href) ")";
    }
    
    /* Evita interruzioni di pagina indesiderate */
    h1, h2, h3 {
        page-break-after: avoid;
    }
    
    img {
        max-width: 100%;
        page-break-inside: avoid;
    }
    
    /* Controlla interruzioni di pagina */
    .page-break {
        page-break-before: always;
    }
}

11.4 Altre Media Query Utili

/* Orientamento del dispositivo */
@media screen and (orientation: landscape) {
    /* Modalità orizzontale */
}

@media screen and (orientation: portrait) {
    /* Modalità verticale */
}

/* Densità di pixel (Retina displays) */
@media screen and (-webkit-min-device-pixel-ratio: 2),
       screen and (min-resolution: 192dpi) {
    /* Stili per schermi ad alta risoluzione */
}

/* Preferenze utente */
@media (prefers-color-scheme: dark) {
    /* Tema scuro automatico */
    body {
        background-color: #1a1a1a;
        color: #f0f0f0;
    }
}

@media (prefers-reduced-motion: reduce) {
    /* Riduci animazioni per utenti sensibili */
    * {
        animation: none !important;
        transition: none !important;
    }
}
