Lezione 4 - SASS/SCSS

Preprocessore per fogli di stile professionali

Argomento: SASS (Syntactically Awesome Style Sheets), il preprocessore CSS più potente e maturo per scrivere fogli di stile mantenibili e scalabili

Descrizione

Questa lezione introduce SASS, un preprocessore CSS che estende le funzionalità del CSS standard aggiungendo caratteristiche di un linguaggio di programmazione. SASS permette di utilizzare variabili, nesting, mixins, functions e molto altro, rendendo il codice CSS più organizzato, riutilizzabile e facile da mantenere.

SASS si distingue per due sintassi: SASS (indentazione) e SCSS (più simile al CSS). In questo corso utilizzeremo principalmente SCSS, che è più familiare per chi già conosce CSS e più diffusa nell'industria.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Comprendere cos'è SASS e quando utilizzarlo
  • Installare e configurare SASS in un progetto
  • Utilizzare variabili per gestire colori, font e valori riutilizzabili
  • Applicare il nesting per scrivere CSS più leggibile
  • Organizzare il codice con partials e moduli
  • Creare mixins e functions per riutilizzare codice
  • Utilizzare @extend e placeholder per ereditare stili
  • Applicare control directives (if, for, each) per generare CSS dinamicamente
  • Implementare best practices per progetti SASS scalabili
  • Creare un sistema di design completo con SASS

Software Necessario

Prerequisiti

  • Node.js: Ambiente runtime JavaScript (versione 14 o superiore)
  • npm: Package manager (incluso con Node.js)
  • Editor di codice: VS Code, Sublime Text, o simili
  • Terminal/Command Line: Per compilare SASS

Installazione SASS

Dopo aver installato Node.js, eseguire da terminale:

# Installazione globale di SASS
npm install sass

# Verifica installazione
npx sass --version
