üìù Codice SASS del Progetto

Caratteristiche Implementate

  • Variabili organizzate: Colori, spacing, typography in mappe strutturate
  • Mixins responsive: Media queries, flexbox utilities, button styles
  • BEM Methodology: Nesting con parent selector per componenti modulari
  • Functions custom: Spacing system, color utilities
  • @each loops: Generazione automatica varianti colore
  • CSS custom properties: Tema dinamico con SASS + CSS variables
  • Architettura scalabile: Codice organizzato e mantenibile

Struttura SCSS (Esempio Semplificato)

// === VARIABLES ===
$colors: (
  'primary': #3498db,
  'success': #2ecc71,
  'warning': #f39c12,
  'danger': #e74c3c,
  'info': #17a2b8,
  'dark': #2c3e50,
  'light': #ecf0f1
);

$spacing-unit: 8px;
$border-radius: 8px;
$transition-speed: 0.3s;

// === MIXINS ===
@mixin flex-center {
  display: flex;
  align-items: center;
  justify-content: center;
}

@mixin card-shadow {
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  transition: box-shadow $transition-speed;
  
  &:hover {
    box-shadow: 0 4px 16px rgba(0, 0, 0, 0.15);
  }
}

@mixin respond-to($breakpoint) {
  @if $breakpoint == 'tablet' {
    @media (max-width: 992px) { @content; }
  } @else if $breakpoint == 'mobile' {
    @media (max-width: 576px) { @content; }
  }
}

// === FUNCTIONS ===
@function spacing($multiplier) {
  @return $spacing-unit * $multiplier;
}

@function color($name) {
  @return map-get($colors, $name);
}

// === COMPONENTS ===
.dashboard {
  display: grid;
  grid-template-columns: 280px 1fr;
  min-height: 100vh;
  
  @include respond-to('tablet') {
    grid-template-columns: 1fr;
  }
}

.sidebar {
  background: color('dark');
  color: white;
  padding: spacing(3);
  
  &__header {
    margin-bottom: spacing(4);
  }
  
  &__logo {
    font-size: 1.5rem;
    font-weight: bold;
  }
}

.nav-item {
  display: flex;
  align-items: center;
  padding: spacing(2);
  color: rgba(white, 0.8);
  text-decoration: none;
  border-radius: $border-radius;
  transition: all $transition-speed;
  
  &:hover {
    background: rgba(white, 0.1);
    color: white;
  }
  
  &--active {
    background: color('primary');
    color: white;
  }
  
  &__icon {
    margin-right: spacing(2);
  }
}

// === UTILITY GENERATION ===
@each $name, $color in $colors {
  .stat-card--#{$name} {
    .stat-card__icon {
      background: rgba($color, 0.1);
      color: $color;
    }
  }
  
  .badge--#{$name} {
    background: rgba($color, 0.1);
    color: darken($color, 10%);
    border: 1px solid $color;
  }
}
