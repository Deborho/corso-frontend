Lezione 4.2 - Variabili SASS

Memorizza e riutilizza valori in tutto il tuo stylesheet

1. Cosa Sono le Variabili?

Le variabili SASS permettono di memorizzare valori (colori, dimensioni, font, ecc.) e riutilizzarli in tutto il foglio di stile. Se devi cambiare un valore, lo modifichi una sola volta nella variabile e il cambiamento si propaga ovunque.

Vantaggi:

  • Elimina ripetizioni
  • Centralizza i valori del design system
  • Facilita la manutenzione e i cambiamenti
  • Migliora la leggibilità del codice
  • Permette di creare temi facilmente

2. Sintassi delle Variabili

// Dichiarazione variabile con $
$nome-variabile: valore;

// Esempi
$primary-color: #3498db;
$font-size-base: 16px;
$spacing-unit: 8px;

// Utilizzo
.button {
  background: $primary-color;
  font-size: $font-size-base;
  padding: $spacing-unit;
}

Convenzioni naming:

  • Inizia con $
  • Usa kebab-case (parole separate da trattino)
  • Nomi descrittivi: $primary-color invece di $blue
  • Raggruppa variabili correlate: $font-size-small, $font-size-medium, $font-size-large

3. Tipi di Dati

3.1 Numeri

// Numeri interi e decimali
$width: 100;
$height: 75.5;

// Con unità
$font-size: 16px;
$margin: 2rem;
$line-height: 1.5; // unitless
$rotation: 45deg;
$duration: 0.3s;

// Operazioni matematiche
$container-width: 1200px;
$sidebar-width: 300px;
$content-width: $container-width - $sidebar-width; // 900px

.content {
  width: $content-width;
  margin: $margin * 2; // 4rem
  font-size: $font-size + 2px; // 18px
}

3.2 Stringhe

// Con o senza virgolette
$font-family: "Helvetica Neue", Arial, sans-serif;
$font-family-mono: Courier, monospace;
$image-path: "/images/";

// Interpolazione stringa (vedremo dopo)
$side: top;
.box {
  border-#{$side}: 1px solid black; // border-top
}

3.3 Colori

// Diversi formati colore
$color-hex: #3498db;
$color-rgb: rgb(52, 152, 219);
$color-rgba: rgba(52, 152, 219, 0.8);
$color-hsl: hsl(204, 70%, 53%);
$color-name: blue;

// Funzioni colore SASS (built-in)
$primary: #3498db;
$primary-light: lighten($primary, 20%);   // #75b8e7
$primary-dark: darken($primary, 20%);     // #1c5a85
$primary-transparent: rgba($primary, 0.5); // rgba(52, 152, 219, 0.5)

.box {
  background: $primary;
  border: 2px solid darken($primary, 10%);
  box-shadow: 0 0 10px rgba($primary, 0.3);
}

3.4 Booleani

// Valori true/false (usati con @if)
$debug-mode: true;
$enable-animations: false;

@if $debug-mode {
  .debug-info {
    display: block;
    background: yellow;
  }
}

3.5 Null

// Valore nullo (proprietà non generata)
$shadow: null;

.box {
  box-shadow: $shadow; // questa riga non verrà generata nel CSS
}

3.6 Liste

// Liste separate da spazi o virgole
$font-stack: Helvetica, Arial, sans-serif;
$padding-values: 10px 20px 10px 20px;
$colors-list: #3498db, #2ecc71, #e74c3c;

// Utilizzo
.element {
  font-family: $font-stack;
  padding: $padding-values;
}

// Accesso elementi lista (1-based!)
$first-color: nth($colors-list, 1); // #3498db
$list-length: length($colors-list); // 3

3.7 Mappe

// Mappe: coppie chiave-valore (come oggetti)
$colors: (
  primary: #3498db,
  secondary: #2ecc71,
  danger: #e74c3c,
  warning: #f39c12,
  info: #3498db
);

$font-sizes: (
  small: 12px,
  medium: 16px,
  large: 20px,
  xlarge: 24px
);

// Accesso valori con map-get()
.alert {
  background: map-get($colors, danger); // #e74c3c
  font-size: map-get($font-sizes, medium); // 16px
}

// Loop su mappe (vedremo nella lezione 7)
@each $name, $color in $colors {
  .text-#{$name} {
    color: $color;
  }
}

4. Scope delle Variabili

4.1 Variabili Globali

// Definite al livello top del file
$primary-color: #3498db; // Globale

.header {
  background: $primary-color; // Accessibile
}

.footer {
  background: $primary-color; // Accessibile
}

4.2 Variabili Locali

// Definite dentro un blocco (scope locale)
.sidebar {
  $sidebar-width: 250px; // Locale a .sidebar
  width: $sidebar-width;  // Accessibile
  
  .sidebar-content {
    width: $sidebar-width; // Accessibile (nested)
  }
}

.main {
  width: $sidebar-width; // ❌ ERRORE! Non accessibile
}

4.3 Shadowing (Sovrascrittura)

$color: red; // Globale

.box {
  color: $color; // red
}

.special-box {
  $color: blue; // Locale (shadowing)
  color: $color; // blue (usa variabile locale)
  
  .inner {
    color: $color; // blue (usa variabile locale del genitore)
  }
}

.another-box {
  color: $color; // red (variabile globale non modificata)
}

4.4 Flag !global

$color: red; // Globale

.box {
  $color: blue !global; // Modifica la variabile globale!
  color: $color; // blue
}

.another-box {
  color: $color; // blue (globale è stata modificata)
}

// ⚠️ Usa !global con cautela! Può creare confusione

5. Flag !default

Il flag !default assegna un valore solo se la variabile non è già stata definita. Utile per creare valori di default personalizzabili.

// Esempio senza !default
$primary-color: #3498db;
$primary-color: #e74c3c; // Sovrascrive
// Risultato: $primary-color è #e74c3c

// Esempio con !default
$primary-color: #3498db;
$primary-color: #e74c3c !default; // NON sovrascrive perché già definita
// Risultato: $primary-color è ancora #3498db

// Caso d'uso pratico: libreria personalizzabile
// File: _defaults.scss (libreria)
$button-bg: #3498db !default;
$button-padding: 10px 20px !default;

.button {
  background: $button-bg;
  padding: $button-padding;
}

// File: main.scss (utente)
// Personalizza prima di importare
$button-bg: #e74c3c; // Sovrascrive il default!
@import 'defaults';

6. Best Practices per Variabili

6.1 Organizzazione Variabili

// File: _variables.scss
// Organizza variabili per categoria

// === COLORS ===
// Brand Colors
$color-primary: #3498db;
$color-secondary: #2ecc71;
$color-accent: #f39c12;

// Semantic Colors
$color-success: #2ecc71;
$color-danger: #e74c3c;
$color-warning: #f39c12;
$color-info: #3498db;

// Neutral Colors
$color-dark: #2c3e50;
$color-grey: #95a5a6;
$color-light: #ecf0f1;
$color-white: #ffffff;

// === TYPOGRAPHY ===
$font-primary: 'Helvetica Neue', Arial, sans-serif;
$font-secondary: Georgia, serif;
$font-mono: 'Courier New', monospace;

$font-size-xs: 12px;
$font-size-sm: 14px;
$font-size-base: 16px;
$font-size-lg: 18px;
$font-size-xl: 24px;
$font-size-xxl: 32px;

$font-weight-light: 300;
$font-weight-normal: 400;
$font-weight-bold: 700;

// === SPACING ===
// Sistema spacing 8px
$spacing-xs: 4px;   // 0.5 unit
$spacing-sm: 8px;   // 1 unit
$spacing-md: 16px;  // 2 units
$spacing-lg: 24px;  // 3 units
$spacing-xl: 32px;  // 4 units
$spacing-xxl: 48px; // 6 units

// === BREAKPOINTS ===
$breakpoint-mobile: 576px;
$breakpoint-tablet: 768px;
$breakpoint-desktop: 992px;
$breakpoint-wide: 1200px;

// === Z-INDEX ===
$z-index-dropdown: 1000;
$z-index-modal: 1050;
$z-index-tooltip: 1100;

// === TRANSITIONS ===
$transition-speed-fast: 0.15s;
$transition-speed-base: 0.3s;
$transition-speed-slow: 0.5s;
$transition-easing: ease-in-out;

6.2 Naming Conventions

  • Semantico: $color-primary invece di $blue
  • Descrittivo: $header-height invece di $h
  • Consistente: Usa sempre lo stesso pattern
  • Prefissi: Raggruppa con prefissi ($color-*, $font-*)
  • Suffissi di stato: $button-hover-bg, $input-focus-border

7. Variabili vs CSS Custom Properties

SASS Variables CSS Custom Properties 
$primary: #3498db;

.button {
  background: $primary;
}
 
:root {
  --primary: #3498db;
}

.button {
  background: var(--primary);
}
  • Compilate a build time
  • Non modificabili a runtime
  • Supporto completo a operazioni
  • Non nel DOM
  • Interpretate dal browser
  • Modificabili con JavaScript
  • Cascading e inheritance
  • Supporto browser moderni

Possono anche essere usate entrambe le varianti

// SASS variables per logica di build
$primary-color: #3498db;
$secondary-color: #2ecc71;

// Genera CSS custom properties per runtime
:root {
  --color-primary: #{$primary-color};
  --color-secondary: #{$secondary-color};
  --color-primary-light: #{lighten($primary-color, 20%)};
}

// Usa CSS variables nel codice
.button {
  background: var(--color-primary);
  &:hover {
    background: var(--color-primary-light);
  }
}

// Cambio tema con JavaScript
// document.documentElement.style.setProperty('--color-primary', '#e74c3c');

Punti Chiave

  • Variabili iniziano con $
  • Supportano diversi tipi: numeri, stringhe, colori, liste, mappe
  • Scope: globali (top-level) o locali (dentro blocchi)
  • !default per valori di default personalizzabili
  • !global per modificare variabili globali (usare con cautela)
  • Organizza variabili in file separati (_variables.scss)
  • Usa nomi semantici e convenzioni consistenti
  • Combina SASS variables con CSS custom properties per flessibilità runtime
