Lezione 12 - JOIN

Combinare dati da più tabelle

Argomento: Le JOIN in SQL permettono di combinare righe da due o più tabelle basandosi su una colonna correlata tra di esse.

Descrizione

In questa lezione impareremo a utilizzare le JOIN, uno degli strumenti più potenti di SQL per lavorare con database relazionali. Le JOIN ci permettono di estrarre dati da più tabelle correlate in un'unica query.

Obiettivi di Apprendimento

  • Comprendere il concetto di JOIN e le relazioni tra tabelle
  • Utilizzare INNER JOIN per estrarre solo i dati corrispondenti
  • Utilizzare LEFT JOIN per includere tutti i dati della tabella sinistra
  • Utilizzare RIGHT JOIN per includere tutti i dati della tabella destra
  • Utilizzare CROSS JOIN per il prodotto cartesiano
  • Utilizzare SELF JOIN per confrontare righe nella stessa tabella
  • Combinare più JOIN in una singola query

Tipi di JOIN in MySQL

Tipo JOIN Descrizione
INNER JOIN Restituisce solo le righe che hanno corrispondenze in entrambe le tabelle
LEFT JOIN Restituisce tutte le righe dalla tabella sinistra e le corrispondenze dalla destra
RIGHT JOIN Restituisce tutte le righe dalla tabella destra e le corrispondenze dalla sinistra
CROSS JOIN Restituisce il prodotto cartesiano di entrambe le tabelle
SELF JOIN Una tabella viene unita a se stessa
