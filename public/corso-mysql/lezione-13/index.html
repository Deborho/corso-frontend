Lezione 13 - Subquery (Query Nidificate)

Query all'interno di altre query

Argomento: Le subquery (o query nidificate) sono query SQL inserite all'interno di altre query. Permettono di eseguire operazioni complesse suddividendole in passaggi logici.

Descrizione

In questa lezione impareremo a utilizzare le subquery per creare query complesse e potenti. Le subquery possono essere usate in diverse parti di una query SQL: nel SELECT, nel FROM, nel WHERE e nell'HAVING.

Obiettivi di Apprendimento

  • Comprendere il concetto di subquery e quando utilizzarle
  • Utilizzare subquery nella clausola WHERE
  • Utilizzare subquery con operatori IN, ANY, ALL
  • Utilizzare subquery con EXISTS e NOT EXISTS
  • Utilizzare subquery nella clausola SELECT (scalar subquery)
  • Utilizzare subquery nella clausola FROM (derived tables)
  • Confrontare subquery correlate e non correlate

Tipi di Subquery

Tipo Descrizione Posizione
Scalar Subquery Restituisce un singolo valore SELECT, WHERE
Row Subquery Restituisce una singola riga WHERE
Table Subquery Restituisce una tabella di risultati FROM, IN, EXISTS
Correlated Subquery Dipende dalla query esterna WHERE, SELECT
