Lezione 1 - Introduzione al linguaggio SQL

Fondamenti del linguaggio SQL e introduzione a MySQL

Argomento: Introduzione al linguaggio SQL e al database MySQL

Descrizione

Questa lezione introduce SQL (Structured Query Language), il linguaggio standard per la gestione dei database relazionali. SQL permette di creare, modificare e interrogare database in modo efficiente e strutturato.

MySQL è uno dei sistemi di gestione di database relazionali (RDBMS) più popolari al mondo, utilizzato da aziende come Facebook, Twitter, YouTube e molte altre.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Comprendere cos'è SQL e perché è importante
  • Conoscere i diversi tipi di comandi SQL (DDL, DML, DCL, TCL)
  • Installare e configurare MySQL
  • Utilizzare un client MySQL per eseguire query
