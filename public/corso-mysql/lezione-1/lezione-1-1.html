Lezione 1.1 - Cos'è SQL e MySQL

Fondamenti del linguaggio SQL

1. Introduzione a SQL

SQL (Structured Query Language) è un linguaggio di programmazione standardizzato utilizzato per gestire e manipolare database relazionali. È stato sviluppato negli anni '70 da IBM e oggi è lo standard per la gestione dei database.

1.1 Caratteristiche di SQL

SQL permette di:

  • Creare database e tabelle
  • Inserire nuovi dati
  • Leggere e interrogare i dati esistenti
  • Aggiornare dati esistenti
  • Eliminare dati
  • Gestire permessi e accessi

1.2 Tipi di Comandi SQL

Tipo Descrizione Comandi
DDL (Data Definition Language) Definisce la struttura del database CREATE, ALTER, DROP, TRUNCATE
DML (Data Manipulation Language) Manipola i dati SELECT, INSERT, UPDATE, DELETE
DCL (Data Control Language) Gestisce i permessi GRANT, REVOKE
TCL (Transaction Control Language) Gestisce le transazioni COMMIT, ROLLBACK, SAVEPOINT

2. MySQL

MySQL è un sistema di gestione di database relazionali (RDBMS) open source. È uno dei database più utilizzati al mondo, noto per la sua velocità, affidabilità e facilità d'uso.

2.1 Perché MySQL?

  • Open Source: Gratuito e con una grande community
  • Prestazioni: Veloce e ottimizzato per applicazioni web
  • Scalabilità: Gestisce grandi quantità di dati
  • Compatibilità: Funziona su molti sistemi operativi
  • Sicurezza: Robusto sistema di permessi e crittografia

3. Installazione di MySQL

3.1 Windows

Scarica MySQL Installer da dev.mysql.com e segui la procedura guidata.

3.2 macOS

# Usando Homebrew
brew install mysql

# Avvia il servizio
brew services start mysql

3.3 Linux (Ubuntu/Debian)

# Installazione
sudo apt update
sudo apt install mysql-server

# Avvia il servizio
sudo systemctl start mysql

4. Connessione a MySQL

Per connettersi a MySQL si può usare il terminale o un client grafico come MySQL Workbench.

4.1 Connessione da Terminale

# Connessione come utente root
mysql -u root -p

# Ti verrà chiesta la password

4.2 Primi Comandi

Una volta connessi, possiamo eseguire i primi comandi:

-- Mostra tutti i database
SHOW DATABASES;

-- Mostra la versione di MySQL
SELECT VERSION();

-- Mostra l'utente corrente
SELECT USER();

-- Uscire da MySQL
EXIT;

5. Esempio Finale

Esempio: Prima sessione MySQL

Ecco un esempio completo di una prima sessione MySQL:

-- Connessione (da terminale: mysql -u root -p)

-- Mostra i database esistenti
SHOW DATABASES;

-- Risultato:
-- +--------------------+
-- | Database           |
-- +--------------------+
-- | information_schema |
-- | mysql              |
-- | performance_schema |
-- | sys                |
-- +--------------------+

-- Mostra la versione
SELECT VERSION();

-- Risultato:
-- +-----------+
-- | VERSION() |
-- +-----------+
-- | 8.0.35    |
-- +-----------+

-- Esci
EXIT;
