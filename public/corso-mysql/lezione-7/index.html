Lezione 7 - Manipolazione colonne

Modificare la struttura delle tabelle con ALTER TABLE

Argomento: ALTER TABLE per modificare la struttura delle tabelle esistenti

Descrizione

In questa lezione impareremo a modificare la struttura delle tabelle esistenti: aggiungere, modificare, rinominare e eliminare colonne.

Obiettivi di Apprendimento

  • Aggiungere nuove colonne con ADD
  • Aggiungere chiavi primarie su tabelle esistenti
  • Posizionare colonne con AFTER e FIRST
  • Rinominare colonne con RENAME COLUMN
  • Modificare il tipo di colonna con MODIFY
  • Eliminare colonne con DROP COLUMN

Argomenti Trattati

  • ALTER TABLE ADD
  • ALTER TABLE ADD PRIMARY KEY
  • AFTER e FIRST
  • ALTER TABLE RENAME COLUMN
  • ALTER TABLE MODIFY
  • ALTER TABLE DROP COLUMN
