Lezione 2.1 - CREATE DATABASE e USE

Creazione e selezione di database

1. CREATE DATABASE

Il comando CREATE DATABASE viene utilizzato per creare un nuovo database in MySQL. Un database è un contenitore che organizza le tabelle e i dati correlati.

1.1 Sintassi Base

CREATE DATABASE nome_database;

1.2 Esempio

-- Crea un nuovo database chiamato "negozio"
CREATE DATABASE negozio;

-- Verifica che il database sia stato creato
SHOW DATABASES;

Risultato:

Query OK, 1 row affected (0.01 sec)

1.3 IF NOT EXISTS

L'opzione IF NOT EXISTS previene errori se il database esiste già.

-- Crea il database solo se non esiste già
CREATE DATABASE IF NOT EXISTS negozio;

1.4 Specificare il Character Set

È buona pratica specificare il character set (codifica dei caratteri) per supportare caratteri speciali:

-- Crea database con supporto per caratteri italiani e emoji
CREATE DATABASE negozio
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;

2. SHOW DATABASES

Il comando SHOW DATABASES elenca tutti i database presenti nel server MySQL.

SHOW DATABASES;

Risultato:

+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| negozio            |
| performance_schema |
| sys                |
+--------------------+

3. USE

Il comando USE seleziona un database come database attivo. Tutte le operazioni successive verranno eseguite su questo database.

3.1 Sintassi

USE nome_database;

3.2 Esempio

-- Seleziona il database "negozio"
USE negozio;

-- Verifica quale database è selezionato
SELECT DATABASE();

Risultato:

+------------+
| DATABASE() |
+------------+
| negozio    |
+------------+

4. DROP DATABASE

Il comando DROP DATABASE elimina un database e tutti i suoi contenuti. Attenzione: questa operazione è irreversibile!

4.1 Sintassi

DROP DATABASE nome_database;

4.2 IF EXISTS

L'opzione IF EXISTS previene errori se il database non esiste:

-- Elimina il database solo se esiste
DROP DATABASE IF EXISTS negozio_test;

5. Esempio Finale

Esempio: Gestione completa di un database

Creiamo e gestiamo un database per un negozio online:

-- 1. Mostra i database esistenti
SHOW DATABASES;

-- 2. Crea un nuovo database
CREATE DATABASE IF NOT EXISTS ecommerce
CHARACTER SET utf8mb4
COLLATE utf8mb4_unicode_ci;

-- 3. Verifica la creazione
SHOW DATABASES;

-- 4. Seleziona il database
USE ecommerce;

-- 5. Verifica la selezione
SELECT DATABASE();

-- Risultato: ecommerce

-- 6. (Opzionale) Elimina il database
-- DROP DATABASE IF EXISTS ecommerce;

Note importanti:

  • I nomi dei database sono case-sensitive su Linux, ma non su Windows
  • Evita spazi e caratteri speciali nei nomi dei database
  • Usa underscore (_) per separare le parole
  • Sempre eseguire backup prima di DROP DATABASE
