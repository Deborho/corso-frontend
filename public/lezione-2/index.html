Lezione 2 - CSS

Stili e presentazione delle pagine web

Argomento: CSS (Cascading Style Sheets) - il linguaggio per lo styling e il layout delle pagine web

Descrizione

Questa lezione si concentra sul CSS (Cascading Style Sheets), il linguaggio utilizzato per controllare la presentazione visuale delle pagine web. Impareremo come applicare stili agli elementi HTML, creare layout complessi e rendere le pagine web esteticamente piacevoli e responsive.

Esploreremo le diverse tecniche di styling, dai selettori base alle proprietà avanzate, fino ai moderni sistemi di layout come Flexbox e Grid.

Obiettivi di Apprendimento

Al termine di questa lezione saremo in grado di:

  • Comprendere la sintassi CSS e le diverse modalità di integrazione
  • Utilizzare i selettori CSS per targettizzare elementi specifici
  • Applicare proprietà CSS per controllare colori, font e spaziature
  • Comprendere e utilizzare il Box Model
  • Gestire il posizionamento degli elementi (static, relative, absolute, fixed)
  • Creare layout responsive con Flexbox
  • Creare layout bidimensionali con CSS Grid
  • Utilizzare pseudo-classi e pseudo-elementi
  • Applicare media queries per il responsive design
  • Utilizzare variabili CSS per una migliore manutenibilità
