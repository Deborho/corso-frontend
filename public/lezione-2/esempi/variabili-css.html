Variabili CSS (Custom Properties)

Cosa sono le Variabili CSS?

Le variabili CSS (custom properties) permettono di memorizzare valori riutilizzabili in un unico posto, facilitando la manutenzione e la coerenza del codice.

Definizione delle Variabili

:root {
  --primary-color: #3498db;
  --spacing-medium: 20px;
  --border-radius: 8px;
}

Utilizzo delle Variabili

.button {
  background-color: var(--primary-color);
  padding: var(--spacing-medium);
  border-radius: var(--border-radius);
}

Vantaggi delle Variabili CSS

  • Manutenibilità: Cambia un valore in un solo posto
  • Coerenza: Garantisce l'uso uniforme di colori, spaziature, ecc.
  • Temi dinamici: Permette di cambiare temi facilmente
  • Leggibilità: Nomi significativi invece di valori hardcoded
  • Scoping: Possono essere ridefinite a livello locale

Nota sul Supporto Browser

Le variabili CSS sono supportate da tutti i browser moderni (Chrome, Firefox, Safari, Edge). Per supporto legacy, considera l'uso di fallback o preprocessori CSS.
