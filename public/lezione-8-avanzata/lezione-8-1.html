Livelli di Configurazione

Git ha tre livelli di configurazione:

1. System (--system)
   - Applica a tutti gli utenti del sistema
   - File: /etc/gitconfig (Linux) o C:\ProgramData\Git\config (Windows)
   - Uso: git config --system [...]

2. Global (--global)
   - Applica a tutti i repository dell'utente corrente
   - File: ~/.gitconfig o ~/.config/git/config
   - Uso: git config --global [...]
   - üìå Pi√π comune

3. Local (--local)
   - Applica solo al repository corrente
   - File: .git/config nella directory del repository
   - Uso: git config --local [...] (default se ometti flag)
   
Priorit√†: Local > Global > System

Configurazioni Utili Aggiuntive

# Configurazione merge tool:

git config --global mergetool.vscode.cmd 'code --wait $MERGED'
