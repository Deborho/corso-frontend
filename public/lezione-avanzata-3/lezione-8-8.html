Setup Fork - Passo per Passo

# 1. Fork repository su GitHub:
# Vai su https://github.com/original-owner/repo
# Click "Fork" button
# Ora hai: https://github.com/tuo-username/repo

# 2. Clone TUO fork:
git clone https://github.com/tuo-username/repo.git
cd repo

# 3. Aggiungi upstream (repository originale):
git remote add upstream https://github.com/original-owner/repo.git

# 4. Verifica remotes:
git remote -v
# origin    https://github.com/tuo-username/repo.git (fetch)
# origin    https://github.com/tuo-username/repo.git (push)
# upstream  https://github.com/original-owner/repo.git (fetch)
# upstream  https://github.com/original-owner/repo.git (push)

# 5. Fetch upstream per aggiornamenti:
git fetch upstream

# 6. Merge aggiornamenti upstream nel tuo main:
git checkout main
git merge upstream/main

# 7. Push aggiornamenti al tuo fork:
git push origin main

Contribuire con Pull Request

# 1. Sincronizza con upstream:
git checkout main
git fetch upstream
git merge upstream/main

# 2. Crea branch per feature:
git checkout -b feature/add-dark-mode

# 3. Lavora sulla feature:
echo "dark mode code" > darkmode.js
git add darkmode.js
git commit -m "feat: add dark mode toggle"

# 4. Push branch al TUO fork:
git push origin feature/add-dark-mode

# 5. Apri Pull Request su GitHub:
# - Vai al repository originale
# - Click "Pull requests" â†’ "New pull request"
# - Click "compare across forks"
# - Base: original-owner/repo main
# - Head: tuo-username/repo feature/add-dark-mode
# - Click "Create pull request"
# - Scrivi descrizione chiara
# - Click "Create pull request"

# 6. Aspetta review e feedback

# 7. Se richieste modifiche:
# Modifica file localmente
git add .
git commit -m "fix: address review comments"
git push origin feature/add-dark-mode
# PR si aggiorna automaticamente!

# 8. Dopo merge:
# - PR chiusa
# - Cancella branch locale e remoto
git checkout main
git branch -d feature/add-dark-mode
git push origin --delete feature/add-dark-mode
