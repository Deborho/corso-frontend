Lezione 3.7 - Componenti Avanzati

Modal, Carousel, Accordion, Tooltips e Componenti JavaScript

1. Modal

I modal sono finestre di dialogo che si sovrappongono al contenuto principale. Richiedono l'interazione dell'utente prima di tornare al contenuto principale.

1.1 Modal Base

<!-- Button trigger modal -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" 
        data-bs-target="#exampleModal">
    Launch demo modal
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">Modal title</h5>
                <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body">
                <p>Modal body text goes here.</p>
            </div>
            <div class="modal-footer">
                <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">
                    Close
                </button>
                <button type="button" class="btn btn-primary">Save changes</button>
            </div>
        </div>
    </div>
</div>

1.2 Modal Sizes

<!-- Small modal -->
<div class="modal-dialog modal-sm">
    ...
</div>

<!-- Default modal -->
<div class="modal-dialog">
    ...
</div>

<!-- Large modal -->
<div class="modal-dialog modal-lg">
    ...
</div>

<!-- Extra large modal -->
<div class="modal-dialog modal-xl">
    ...
</div>

<!-- Fullscreen modal -->
<div class="modal-dialog modal-fullscreen">
    ...
</div>

1.3 Modal Scrollable

<div class="modal-dialog modal-dialog-scrollable">
    <div class="modal-content">
        <div class="modal-header">
            <h5 class="modal-title">Modal title</h5>
            <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
        </div>
        <div class="modal-body">
            <p>Long content...</p>
            <!-- Contenuto lungo che causerà scroll -->
        </div>
        <div class="modal-footer">
            <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">
                Close
            </button>
        </div>
    </div>
</div>

1.4 Modal Centered

<div class="modal-dialog modal-dialog-centered">
    <div class="modal-content">
        ...
    </div>
</div>

1.5 Modal con JavaScript

<script>
// Mostra modal
var myModal = new bootstrap.Modal(document.getElementById('myModal'))
myModal.show()

// Nascondi modal
myModal.hide()

// Toggle modal
myModal.toggle()

// Eventi
var modalElement = document.getElementById('myModal')
modalElement.addEventListener('show.bs.modal', function (event) {
    console.log('Modal sta per essere mostrato')
})
modalElement.addEventListener('shown.bs.modal', function (event) {
    console.log('Modal è stato mostrato')
})
modalElement.addEventListener('hide.bs.modal', function (event) {
    console.log('Modal sta per essere nascosto')
})
modalElement.addEventListener('hidden.bs.modal', function (event) {
    console.log('Modal è stato nascosto')
})
</script>

<div class="carousel slide" data-bs-ride="carousel" 
     data-bs-interval="3000" data-bs-pause="hover">

<script>
var myCarousel = new bootstrap.Carousel(document.getElementById('myCarousel'), {
    interval: 3000,  // Tempo tra gli slide (ms)
    pause: 'hover',  // Pausa on hover
    wrap: true,      // Continua ciclicamente
    keyboard: true   // Controlla con tastiera
})
</script>

3. Accordion

L'accordion permette di mostrare/nascondere contenuto in sezioni collapsabili. Utile per FAQ, menu di navigazione o contenuti organizzati.

             data-bs-parent="#accordionExample">
            <div class="accordion-body">
                Content for accordion item #1
            </div>
        </div>
    </div>
    
    <div class="accordion-item">

                    data-bs-toggle="collapse" data-bs-target="#collapseTwo">
                Accordion Item #2
            </button>
        </h2>
        <div id="collapseTwo" class="accordion-collapse collapse" 
             data-bs-parent="#accordionExample">
            <div class="accordion-body">
                Content for accordion item #2
            </div>
        </div>
    </div>
    
    <div class="accordion-item">

4. Collapse

Il componente collapse permette di mostrare/nascondere contenuto. È la base per accordion e può essere usato indipendentemente.

4.1 Collapse Base

<!-- Trigger con button -->
<button class="btn btn-primary" type="button" data-bs-toggle="collapse" 
        data-bs-target="#collapseExample">
    Toggle content
</button>

<!-- Trigger con link -->
<a class="btn btn-primary" data-bs-toggle="collapse" href="#collapseExample">
    Toggle content
</a>

<!-- Content -->
<div class="collapse" id="collapseExample">
    <div class="card card-body">
        Some placeholder content for the collapse component.
    </div>
</div>

4.2 Multiple Targets

<button class="btn btn-primary" type="button" data-bs-toggle="collapse" 
        data-bs-target=".multi-collapse">
    Toggle both elements
</button>

<div class="collapse multi-collapse" id="multiCollapseExample1">
    <div class="card card-body">
        Content 1
    </div>
</div>

<div class="collapse multi-collapse" id="multiCollapseExample2">
    <div class="card card-body">
        Content 2
    </div>
</div>

4.3 Horizontal Collapse

<button class="btn btn-primary" type="button" data-bs-toggle="collapse" 
        data-bs-target="#collapseWidthExample">
    Toggle width collapse

<div style="min-height: 120px;">

5. Tooltips

I tooltips sono piccoli popup che appaiono al passaggio del mouse. Devono essere inizializzati con JavaScript.

5.1 Tooltip Base

<!-- HTML -->
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" 
        data-bs-placement="top" title="Tooltip on top">
    Tooltip on top
</button>

<!-- Inizializzazione JavaScript (richiesto) -->
<script>
var tooltipTriggerList = [].slice.call(
    document.querySelectorAll('[data-bs-toggle="tooltip"]')
)
var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
    return new bootstrap.Tooltip(tooltipTriggerEl)
})
</script>

5.2 Tooltip Placement

<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" 
        data-bs-placement="top" title="Tooltip on top">
    Tooltip on top
</button>

<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" 
        data-bs-placement="right" title="Tooltip on right">
    Tooltip on right
</button>

<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" 
        data-bs-placement="bottom" title="Tooltip on bottom">
    Tooltip on bottom
</button>

<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" 
        data-bs-placement="left" title="Tooltip on left">
    Tooltip on left
</button>

5.3 Tooltip con HTML

<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" 
        data-bs-html="true" 
        title="<em>Tooltip</em> <u>with</u> <b>HTML</b>">
    Tooltip with HTML
</button>

6. Popovers

I popover sono simili ai tooltip ma più grandi e con più contenuto. Possono contenere titolo e body. Richiedono inizializzazione JavaScript.

6.1 Popover Base

<!-- HTML -->
<button type="button" class="btn btn-lg btn-danger" data-bs-toggle="popover" 
        title="Popover title" data-bs-content="And here's some amazing content.">
    Click to toggle popover
</button>

<!-- Inizializzazione JavaScript (richiesto) -->
<script>
var popoverTriggerList = [].slice.call(
    document.querySelectorAll('[data-bs-toggle="popover"]')
)
var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl)
})
</script>

6.2 Popover Directions

<button type="button" class="btn btn-secondary" data-bs-container="body" 
        data-bs-toggle="popover" data-bs-placement="top" 
        data-bs-content="Top popover">
    Popover on top
</button>

<button type="button" class="btn btn-secondary" data-bs-container="body" 
        data-bs-toggle="popover" data-bs-placement="right" 
        data-bs-content="Right popover">
    Popover on right
</button>

<button type="button" class="btn btn-secondary" data-bs-container="body" 
        data-bs-toggle="popover" data-bs-placement="bottom" 
        data-bs-content="Bottom popover">
    Popover on bottom
</button>

<button type="button" class="btn btn-secondary" data-bs-container="body" 
        data-bs-toggle="popover" data-bs-placement="left" 
        data-bs-content="Left popover">
    Popover on left
</button>

6.3 Dismissible Popover

<button type="button" class="btn btn-lg btn-danger" data-bs-toggle="popover" 
        data-bs-trigger="focus" title="Dismissible popover" 
        data-bs-content="Click outside to dismiss">
    Dismissible popover
</button>

7. Offcanvas

Offcanvas crea sidebar nascoste che possono scorrere da qualsiasi lato dello schermo. Perfette per menu mobili o pannelli laterali.

7.1 Offcanvas Base

<!-- Trigger button -->
<button class="btn btn-primary" type="button" data-bs-toggle="offcanvas" 
        data-bs-target="#offcanvasExample">
    Toggle offcanvas
</button>

<!-- Offcanvas -->
<div class="offcanvas offcanvas-start" tabindex="-1" id="offcanvasExample">
    <div class="offcanvas-header">
        <h5 class="offcanvas-title">Offcanvas</h5>
        <button type="button" class="btn-close" data-bs-dismiss="offcanvas"></button>
    </div>
    <div class="offcanvas-body">
        <div>
            Some text as placeholder.
        </div>
    </div>
</div>

7.2 Offcanvas Placement

<!-- Left (start) -->
<div class="offcanvas offcanvas-start" id="offcanvasLeft">...</div>

<!-- Right (end) -->
<div class="offcanvas offcanvas-end" id="offcanvasRight">...</div>

<!-- Top -->
<div class="offcanvas offcanvas-top" id="offcanvasTop">...</div>

<!-- Bottom -->
<div class="offcanvas offcanvas-bottom" id="offcanvasBottom">...</div>

7.3 Offcanvas con Backdrop

<!-- Default: backdrop + scrolling bloccato -->
<button data-bs-toggle="offcanvas" data-bs-target="#offcanvas1">
    Default offcanvas
</button>

<!-- Con scrolling body abilitato -->
<button data-bs-toggle="offcanvas" data-bs-target="#offcanvas2" 
        data-bs-scroll="true">
    Enable body scrolling
</button>

<!-- Senza backdrop -->
<button data-bs-toggle="offcanvas" data-bs-target="#offcanvas3" 
        data-bs-backdrop="false">
    No backdrop
</button>

8. Toast

I toast sono notifiche leggere e non invasive che appaiono temporaneamente. Perfetti per feedback dell'utente.

8.1 Toast Base

<div class="toast" role="alert">
    <div class="toast-header">
        <strong class="me-auto">Bootstrap</strong>
        <small>11 mins ago</small>
        <button type="button" class="btn-close" data-bs-dismiss="toast"></button>
    </div>
    <div class="toast-body">
        Hello, world! This is a toast message.
    </div>
</div>

<!-- Attivazione JavaScript -->
<script>
var toastEl = document.getElementById('myToast')
var toast = new bootstrap.Toast(toastEl)
toast.show()
</script>

8.2 Toast Placement

<!-- Container per posizionamento -->
<div class="toast-container position-fixed top-0 end-0 p-3">
    <div class="toast" role="alert">
        <div class="toast-body">
            Toast in alto a destra
        </div>
    </div>
</div>

<!-- Altre posizioni -->
<!-- top-0 start-0 = alto sinistra -->
<!-- top-0 end-0 = alto destra -->
<!-- bottom-0 start-0 = basso sinistra -->
<!-- bottom-0 end-0 = basso destra -->

8.3 Toast con Color

<div class="toast align-items-center text-bg-primary border-0" role="alert">
    <div class="d-flex">
        <div class="toast-body">
            Primary toast
        </div>
        <button type="button" class="btn-close btn-close-white me-2 m-auto" 
                data-bs-dismiss="toast"></button>
    </div>
</div>

<!-- Varianti: text-bg-success, text-bg-danger, text-bg-warning, etc. -->

10. Best Practices

✓ Consigli:

  • Inizializza tooltip e popover con JavaScript prima di usarli
  • Usa modal per azioni importanti che richiedono attenzione
  • Usa offcanvas per menu di navigazione su mobile
  • Usa toast per notifiche non invasive
  • Accordion perfetto per FAQ e contenuti organizzati
  • Carousel ideale per gallerie ma usa con moderazione
  • Fornisci sempre modi alternativi per chiudere modal/offcanvas (ESC, click outside)

✗ Evita:

  • Non abusare di modal - possono essere invasivi
  • Non usare carousel auto-play per contenuti critici
  • Non bloccare l'utente con modal senza via d'uscita
  • Non usare troppi tooltip - possono confondere
