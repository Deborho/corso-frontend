Lezione 3.3 - Typography e Utilities

Classi utility Bootstrap per tipografia, spacing, colori e layout

1. Typography (Tipografia)

Bootstrap fornisce un sistema tipografico completo con classi per headings, display text, paragrafi e formattazione del testo. Tutte le classi sono responsive e seguono best practices di tipografia web.

1.1 Headings (Intestazioni)

Bootstrap stila automaticamente i tag HTML <h1> fino a <h6>. Puoi anche usare le classi .h1 fino a .h6 per applicare lo stile senza usare il tag semantico:

<!-- Tag HTML semantici -->
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<!-- Classi per stile senza semantica -->
<p class="h1">Stile h1 su un paragrafo</p>
<div class="h3">Stile h3 su un div</div>

1.2 Display Headings

Per titoli ancora più grandi e impattanti:

<h1 class="display-1">Display 1</h1>
<h1 class="display-2">Display 2</h1>
<h1 class="display-3">Display 3</h1>
<h1 class="display-4">Display 4</h1>
<h1 class="display-5">Display 5</h1>
<h1 class="display-6">Display 6</h1>
Classe Dimensione (rem) Uso
.display-1 5rem (80px) Hero sections, titoli principali
.display-2 4.5rem (72px) Titoli molto grandi
.display-3 4rem (64px) Titoli grandi
.display-4 3.5rem (56px) Sottotitoli importanti

1.3 Lead Paragraph

Per far risaltare un paragrafo:

<p class="lead">
    Questo è un paragrafo lead, più grande e con maggior peso 
    visuale rispetto ai paragrafi normali.
</p>

4. Spacing Utilities

Bootstrap usa un sistema di spacing basato su una scala da 0 a 5 (più auto). Ogni livello corrisponde a un multiplo di 0.25rem (4px di default).

Formato: {property}{sides}-{size} o {property}{sides}-{breakpoint}-{size}

4.1 Properties

Property Significato
m margin
p padding

4.2 Sides

Side Significato
t top
b bottom
s start (left in LTR, right in RTL)
e end (right in LTR, left in RTL)
x left e right
y top e bottom
(nessuno) tutti e 4 i lati

4.3 Sizes

Size Valore Pixel (default)
0 0 0px
1 0.25rem 4px
2 0.5rem 8px
3 1rem 16px
4 1.5rem 24px
5 3rem 48px
auto auto auto

4.4 Esempi Spacing

<div class="mt-4">Margin top (24px)</div>
<div class="mb-2">Margin bottom (8px)</div>
<div class="ms-5">Margin start/left (48px)</div>
<div class="me-1">Margin end/right (4px)</div>
<div class="mx-3">Margin left e right (16px)</div>
<div class="my-4">Margin top e bottom (24px)</div>
<div class="mx-auto">Margin auto left e right (centra)</div>

<!-- Padding -->
<div class="p-3">Padding su tutti i lati (16px)</div>
<div class="pt-4">Padding top (24px)</div>
<div class="pb-2">Padding bottom (8px)</div>
<div class="ps-5">Padding start/left (48px)</div>
<div class="pe-1">Padding end/right (4px)</div>
<div class="px-3">Padding left e right (16px)</div>
<div class="py-4">Padding top e bottom (24px)</div>

<!-- Margin negativo (solo margin) -->

    Mobile: 8px | Tablet: 24px | Desktop: 48px

12. Best Practices

✓ Consigli:

  • Usa utilities per prototipazione rapida
  • Preferisci utilities a CSS custom per spacing e colori
  • Combina utilities per effetti complessi
  • Usa classi responsive quando necessario
  • Mantieni consistenza nei valori di spacing

✗ Evita:

  • Non sovrascrivere utilities con !important
  • Non abusare delle utilities (troppe classi su un elemento)
  • Non mischiare utilities Bootstrap con altri framework CSS
  • Non usare valori di spacing custom se possibile
