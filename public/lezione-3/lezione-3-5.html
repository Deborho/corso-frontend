Lezione 3.5 - Componenti Navigazione

Navbar, Navs, Tabs, Breadcrumb e Pagination

1. Navbar

La navbar di Bootstrap è un componente responsive per creare header di navigazione. Include supporto per branding, navigazione, form e altro ancora. Si adatta automaticamente ai diversi dispositivi con un menu hamburger su mobile.

1.1 Navbar Base

<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Navbar</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" 
                data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav">
                <li class="nav-item">
                    <a class="nav-link active" href="#">Home</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">Features</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">Pricing</a>
                </li>
            </ul>
        </div>
    </div>
</nav>

1.2 Navbar con Brand e Logo

<nav class="navbar navbar-light bg-light">
    <div class="container">
        <!-- Brand con testo -->
        <a class="navbar-brand" href="#">Navbar</a>
        
        <!-- Brand con immagine -->
        <a class="navbar-brand" href="#">
            <img src="/logo.svg" alt="Logo" width="30" height="24">
        </a>
        
        <!-- Brand con immagine e testo -->
        <a class="navbar-brand" href="#">
            <img src="/logo.svg" alt="" width="30" height="24" 
                 class="d-inline-block align-text-top">
            Bootstrap
        </a>
    </div>
</nav>

1.7 Navbar Fixed

<!-- Fixed to top -->

    <div class="container-fluid">
        <a class="navbar-brand" href="#">Fixed top</a>
    </div>
</nav>

<!-- Fixed to bottom -->
<nav class="navbar navbar-light bg-light fixed-bottom">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Fixed bottom</a>
    </div>
</nav>

<!-- Sticky top -->
<nav class="navbar navbar-light bg-light sticky-top">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Sticky top</a>
    </div>
</nav>

Nota: Quando usi .fixed-top o .fixed-bottom, ricorda di aggiungere padding al body per evitare che il contenuto venga coperto dalla navbar.

1.8 Offcanvas Navbar

Per mobile, usa offcanvas invece del collapse standard:

<nav class="navbar navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Offcanvas navbar</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="offcanvas" 
                data-bs-target="#offcanvasNavbar">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="offcanvas offcanvas-end" tabindex="-1" id="offcanvasNavbar">
            <div class="offcanvas-header">
                <h5 class="offcanvas-title">Menu</h5>
                <button type="button" class="btn-close" data-bs-dismiss="offcanvas"></button>
            </div>
            <div class="offcanvas-body">
                <ul class="navbar-nav">
                    <li class="nav-item">
                        <a class="nav-link active" href="#">Home</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#">Link</a>
                    </li>
                </ul>
            </div>
        </div>
    </div>
</nav>

3. Breadcrumb

Il breadcrumb è un indicatore della posizione corrente all'interno della gerarchia di navigazione del sito. Utile per aiutare gli utenti a capire dove si trovano.

3.1 Breadcrumb Base

<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="#">Home</a></li>
        <li class="breadcrumb-item active" aria-current="page">Library</li>
    </ol>
</nav>

<nav aria-label="breadcrumb">
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="#">Home</a></li>
        <li class="breadcrumb-item"><a href="#">Library</a></li>
        <li class="breadcrumb-item active" aria-current="page">Data</li>
    </ol>
</nav>

3.2 Breadcrumb con Separatore Custom

<style>
    .breadcrumb-custom {
        --bs-breadcrumb-divider: '>';
    }
    
    .breadcrumb-arrow {
        --bs-breadcrumb-divider: '→';
    }
</style>

<nav aria-label="breadcrumb">
    <ol class="breadcrumb breadcrumb-custom">
        <li class="breadcrumb-item"><a href="#">Home</a></li>
        <li class="breadcrumb-item active">Library</li>
    </ol>
</nav>

4. Pagination

La pagination permette di navigare tra pagine di contenuto. Bootstrap fornisce stili per creare controlli di paginazione accessibili e responsive.

4.1 Pagination Base

<nav aria-label="Page navigation">
    <ul class="pagination">
        <li class="page-item"><a class="page-link" href="#">Previous</a></li>
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
        <li class="page-item"><a class="page-link" href="#">Next</a></li>
    </ul>
</nav>

6. Best Practices

✓ Consigli:

  • Usa sempre navbar-expand-{breakpoint} per controllo responsive
  • Includi sempre aria-label e attributi accessibilità
  • Usa .active per indicare la pagina corrente
  • Per fixed navbar, aggiungi padding al body
  • Usa breadcrumb per siti con gerarchia profonda
  • Pagination sempre con <nav> e aria-label

✗ Evita:

  • Non usare troppe voci nella navbar (max 7-8)
  • Non dimenticare il toggler button per mobile
  • Non usare dropdown troppo complessi
  • Non omettere gli attributi accessibilità
  • Non usare pagination per poche pagine (usa scroll)
