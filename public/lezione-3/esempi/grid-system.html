Esempio 1: Mobile → Tablet → Desktop

7. Order (Ordine Visuale)

Cambia l'ordine visuale delle colonne senza modificare l'HTML

Order Responsive (utile per mobile)

<div class="order-2 order-lg-1">Main</div> <div class="order-1 order-lg-2">Sidebar</div>

Mobile: Sidebar prima | Desktop: Main prima

Contenuto Principale
order-2 order-lg-1
Su mobile appare DOPO la sidebar
Sidebar
order-1 order-lg-2
Su mobile appare PRIMA del main
